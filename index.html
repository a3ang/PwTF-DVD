Beyond Spatial Frequency: Pixel-wise Temporal Frequency-based Deepfake Video Detection [ICCV 2025]

Taehoon Kim, Jongwook Choi, Yonghyun Jeang, Haeun Noh, Jaejun Yoo, Seungryul Baek, Jongwon Choi

Overview

This project captures subtle temporal artifacts by applying a 1D Fourier transform to each pixel over time, then focuses on anomalous regions using an attention module for robust deepfake video detection.
