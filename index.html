PwTF-DVD

Pixel-wise Temporal Frequency Deepfake Video Detection

Overview

This project captures subtle temporal artifacts by applying a 1D Fourier transform to each pixel over time, then focuses on anomalous regions using an attention module for robust deepfake video detection.

Installation

git clone https://github.com/rama0126/PwTF-DVD.git
cd PwTF-DVD

conda create -n pwTF python=3.10 -y
conda activate pwTF
pip install -r requirements.txt

Usage

# Train
python src/train.py --config configs/train_ffpp.yaml

# Evaluate
python src/evaluate.py --checkpoint results/ffpp_best.pth --dataset cdf --metrics auc eer

# Visualize
jupyter notebook notebooks/temporal_frequency_visualization.ipynb
